# Code of Conduct

## Our standard

This project is meant to preserve care, wonder, and dignity. We expect the same from contributors.

Please be:

- respectful
- clear
- constructive
- privacy-aware

Please do not:

- harass, insult, or demean others
- derail discussion with hostility
- pressure maintainers to expose user data or secrets
- submit contributions that conflict with the project’s core purpose

## Enforcement

Maintainers may remove comments, issues, or contributions that undermine a safe and constructive environment.

Serious or repeated violations may lead to removal from the project space.
